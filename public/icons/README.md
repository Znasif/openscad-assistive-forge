# PWA Icons

This directory contains app icons for Progressive Web App support.

## Official Logo

The project uses the official **OpenSCAD Assistive Web Forge Logo** which features the accessibility symbol overlaid on a 3D model sphere, representing the project's focus on making OpenSCAD accessible to all users.

## Icon Files

### Favicon Sizes
- `favicon-16x16.png` - 16×16px standard favicon
- `favicon-32x32.png` - 32×32px standard favicon
- `icon-16x16.png` - 16×16px icon
- `icon-32x32.png` - 32×32px icon
- `icon-64x64.png` - 64×64px icon

### PWA Icons
- `icon-128x128.png` - 128×128px PWA icon
- `icon-192x192.png` - 192×192px PWA icon (standard)
- `icon-512x512.png` - 512×512px PWA icon (maskable)

### Platform-Specific Icons
- `apple-touch-icon.png` - 180×180px Apple touch icon (iOS home screen)
- `logo.png` - Large format logo for general use
- `logo.svg` - SVG version (scalable, used as fallback)

### Generated SVG Icons (Auto-generated)
- `icon-*x*.svg` - Various sizes (generated by scripts, can be regenerated)

## Icon Usage

| File | Used For | Dimensions |
|------|----------|------------|
| `favicon-16x16.png` | Browser favicon | 16×16px |
| `favicon-32x32.png` | Browser favicon | 32×32px |
| `apple-touch-icon.png` | iOS home screen, Safari pinned tab | 180×180px |
| `icon-128x128.png` | PWA manifest, Windows tile | 128×128px |
| `icon-192x192.png` | PWA manifest standard icon | 192×192px |
| `icon-512x512.png` | PWA manifest maskable icon | 512×512px |
| `logo.png` | Header logo, fallback icon | Variable |
| `logo.svg` | Scalable vector fallback | Any size |

## Screenshots

The manifest also references screenshots in `/screenshots/`:
- `desktop.png` (1280x720) - Desktop view
- `mobile.png` (750x1334) - Mobile view

These can be captured from the running app using browser dev tools.
