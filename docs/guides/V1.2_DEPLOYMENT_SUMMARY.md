## üì¶ What's Included

### New in v1.2.0
1. **Auto-Preview Controller** (`auto-preview-controller.js` - 375 lines)
   - Progressive quality rendering
   - Debounced auto-render (350ms default)
   - Render caching with LRU eviction
   - Visual state indicators

2. **Enhanced Render Controller**
   - Quality presets (PREVIEW, FULL)
   - `renderPreview()` and `renderFull()` methods
   - Adaptive tier caps for preview quality

3. **Improved Main Application**
   - Auto-preview integration
   - Smart download button logic
   - Rendering overlay with spinner
   - Preview state UI updates

### Carried Forward from v1.1.0
- URL parameter persistence
- localStorage auto-save
- Keyboard shortcuts (Ctrl+Enter, R, D)
- Copy share link button
- Export parameters as JSON
- 3 example models (Simple Box, Parametric Cylinder, Universal Cuff)

### Core Features (v1.0.0)
- OpenSCAD WASM rendering
- Parameter extraction and UI generation
- 3D preview with Three.js
- STL download
- WCAG 2.1 AA accessibility
