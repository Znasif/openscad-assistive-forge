## Known Limitations

1. **No Persistence**: Queue not saved on page refresh
2. **No Parallelization**: Jobs processed sequentially only
3. **Binary Data**: Not included in export (parameters only)
4. **20 Job Limit**: Prevents excessive memory usage
5. **FIFO Only**: No priority queue or reordering
