## ğŸ“ Developer Experience

### Installation
```bash
# Global installation
npm install -g openscad-web-customizer-forge

# Or use with npx
npx openscad-web-customizer-forge --help
```

### Typical Workflow
```bash
# 1. Extract parameters
openscad-forge extract model.scad --out schema.json --pretty

# 2. Generate web app
openscad-forge scaffold \
  --schema schema.json \
  --scad model.scad \
  --out customizer

# 3. Build and test
cd customizer
npm install
npm run dev

# 4. Validate
openscad-forge validate .

# 5. Fix issues
openscad-forge sync . --apply-safe-fixes
```
